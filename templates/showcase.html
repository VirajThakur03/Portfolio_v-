Skillo

AI-Enabled Local Skill Marketplace Platform

Skillo is an AI-enabled local skill marketplace that connects customers with verified service professionals through a secure, real-time booking platform.

The platform features multi-step provider verification, live worker tracking, real-time chat, and role-based dashboards to ensure trust, safety, and transparency at every stage of a job — from booking to completion.

Built with Flask, PostgreSQL, WebSockets, and Docker, this project demonstrates production-grade backend engineering, real-time systems, and scalable marketplace architecture.
